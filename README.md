## BBG

### 3-digit Bulls/Bears/Goats guessing game (console)

**Rules:**
- Secret: a random 3-digit number between 100 and 999, with no repeated digits.
- Player: enters guesses (3-digit, no repeated digits, no leading zero).
- After each guess show counts:
    Bulls  = digits correct and in the correct position
    Bears  = digits present in secret but in a different position
    Goats  = digits not present in secret
- Player gets 5 attempts. Show guess history. Offer "play again" at the end.
